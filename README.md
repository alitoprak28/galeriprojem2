# Velocita Motors

Premium, modern ve dönüşüm odaklı bir araç galerisi web sitesi.

## Teknoloji

- Next.js App Router
- Tailwind CSS
- Framer Motion
- TypeScript

## Sayfalar

- Ana sayfa
- Araç listeleme
- Araç detay
- Markalar
- İletişim

## Çalıştırma

```bash
npm install
npm run dev
```

## Notlar

- Araç verileri `lib/data.ts` içinde örnek yapı ile tanımlanmıştır.
- WhatsApp odaklı lead form akışı önceliklidir.
- Uzak görseller için `images.unsplash.com` kaynağı kullanılmıştır.
